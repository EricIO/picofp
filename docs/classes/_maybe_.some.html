match

  • match<U, V>(matcher: MaybeMatcher<T, U, V>): U | V

  • Inherited from Maybe.match

    • Defined in Maybe.ts:73

    match provides Scala and Rust like facilites to match on the Maybe value. This is similar to the functionality of Maybe.unwrapOr but the value for the None branch can be computed from the function (such as enclosing a value from the parent scope).

    Example:

    const some: Maybe<number> = Some(10);
some.match({
    'Some': (value: T) => return value * 2,
    'None'; () => return 2,
});
